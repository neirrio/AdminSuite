## О проекте

**AdminSuite** — плагин для администрирования Paper-сервера, объединяющий в
одном месте модерацию, экономику, телепорты и десятки утилит, вместо связки
из 5–6 разных плагинов. Цель проекта — **300+ рабочих команд**, разбитых на
9 логических модулей.

> ⚠️ **Статус: v1.0.0 (Beta).** Сейчас готовы 2 из 9 модулей. Остальные
> в активной разработке — прогресс отслеживается в разделе [Roadmap](#-roadmap).

## 🎮 Реализованные команды

<details>
<summary><b>🌍 Мир и погода (20 команд)</b></summary>

| Команда | Описание |
|---|---|
| `/setweather [мир] <clear\|rain\|thunder>` | Устанавливает погоду |
| `/clearsky [мир]` | Очищает небо |
| `/storm [мир]` | Включает дождь |
| `/thunder [мир]` | Включает грозу |
| `/settime [мир] <тики\|day\|night\|noon\|midnight>` | Устанавливает время |
| `/day`, `/night`, `/noon`, `/midnight` | Быстрая смена времени суток |
| `/setworldborder <размер> [x] [z]` | Граница мира |
| `/gamerule <правило> <значение>` | Игровое правило |
| `/spawnmob <тип> [кол-во]` | Спавн существ |
| `/killall [мир] [monsters\|animals\|all]` | Массовое удаление существ |
| `/setspawn`, `/delspawn` | Точка спавна сервера |
| `/setbiome <биом> [радиус]` | Смена биома |
| `/fillarea <материал> <радиус>` | Заливка области блоками |
| `/replaceblock <из> <на> <радиус>` | Замена блоков |
| `/listworlds` | Список миров |
| `/tpworld <мир>` | Телепорт в мир |

</details>

<details>
<summary><b>🧭 Телепорты и навигация (18 команд)</b></summary>

| Команда | Описание |
|---|---|

| `/warp <название>`, `/setwarp`, `/delwarp`, `/warps` | Система варпов |
| `/back` | Вернуться на предыдущую позицию |
| `/rtp` | Случайный безопасный телепорт |
| `/coords` | Текущие координаты |
| `/compass` | Направление взгляда |

</details>
